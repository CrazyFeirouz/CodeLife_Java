2. **Java结构（选择、循环）（structure）**

        + PrimeNumber ： *查找质数(1-100000)

3. **Java数组（arrays）**

    + 数组赋值（assignment）

        + PascalTriangle ：*杨辉三角
            使用二维数组 打印10行 杨辉三角*
            
    + 求数组平均值、最大最小值、和等（value）

    + 数组的复制、反转、查找（线性查找、二分法查找）（crl）
    
        + ArrayReverse : *数组反转*
        
        + ArrayLookup ： *数组查找
            主要写线性查找、二分法查找*
    
    + 数组排序（sort）

       + BubbleSort ： *冒泡排序*
